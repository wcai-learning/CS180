This script aligns images with three vertically stacked color channels (red, green, blue) and outputs aligned RGB images.

Steps:

1. Place '.jpg' and '.tif' files in '/intput' in the current directory

2. Run - python main.py [--crop] [--contrast] [--white_balance]

3. Aligned images will be saved in /output_* folder based on previous arguments
